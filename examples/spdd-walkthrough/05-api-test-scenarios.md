## scenario-001 — within-quota plus overage (normal, op-001)

**Given:** Standard customer `cust-1` with 90,000 fast-model tokens already used in the current period.
**Pricing:** fast-model = $0.01 / 1,000 tokens.

**Setup:**
```bash
curl -X POST $BASE/customers \
  -H 'Content-Type: application/json' \
  -d '{"id":"cust-1","plan":"Standard"}'

curl -X POST $BASE/internal/seed-usage \
  -H 'Content-Type: application/json' \
  -d '{"customerId":"cust-1","tokens":90000,"model":"fast-model","periodStart":"2026-04-01T00:00:00Z"}'
```

**When:**
```bash
curl -X POST $BASE/usage \
  -H 'Content-Type: application/json' \
  -d '{"customerId":"cust-1","tokens":30000,"model":"fast-model","eventId":"evt-1"}' \
  -o /tmp/scenario-001.json -w 'HTTP %{http_code}\n'
```

**Then:**
- HTTP status: `200`.
- Body assertion (jq):
  ```bash
  jq -e '.fromQuota == 10000 and .overage == 20000 and .charged == "0.20" and .currency == "USD"' \
    /tmp/scenario-001.json
  ```

## scenario-003 — unknown model (error, op-001)

**Given:** any Standard customer with quota remaining; the plan has no rule for `weird-model`.

**When:**
```bash
curl -X POST $BASE/usage \
  -H 'Content-Type: application/json' \
  -d '{"customerId":"cust-1","tokens":1,"model":"weird-model","eventId":"evt-3"}' \
  -o /tmp/scenario-003.json -w 'HTTP %{http_code}\n'
```

**Then:**
- HTTP status: `400`.
- Body shape: `{"error": {"code": "UNKNOWN_MODEL", "model": "weird-model"}}`.
- Critically: **no** ledger row exists for `evt-3`:
  ```bash
  curl -s "$BASE/internal/ledger/cust-1?period=2026-04" \
    | jq -e '[.entries[] | select(.eventId == "evt-3")] | length == 0'
  ```

## scenario-004 — replay (boundary, SG-1 idempotency)

**Given:** the result of scenario-001 (so `evt-1` is already in the ledger).

**When:**
```bash
curl -X POST $BASE/usage \
  -H 'Content-Type: application/json' \
  -d '{"customerId":"cust-1","tokens":30000,"model":"fast-model","eventId":"evt-1"}' \
  -o /tmp/scenario-004.json -w 'HTTP %{http_code}\n'
```

**Then:**
- HTTP status: `200`.
- The response is **byte-identical** to scenario-001:
  ```bash
  diff /tmp/scenario-001.json /tmp/scenario-004.json
  ```
- The ledger contains exactly one entry for `evt-1`:
  ```bash
  curl -s "$BASE/internal/ledger/cust-1?period=2026-04" \
    | jq -e '[.entries[] | select(.eventId == "evt-1")] | length == 1'
  ```
